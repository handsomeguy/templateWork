### templateWork 简易模板引擎

- 前言：

自行实现的简易模板引擎，相当于自己的一个小练手项目了，可以参考，不建议项目中使用，因为可能还存在未知bug哈哈哈哈

参考了jQuery tmpl，是由jQuery的作者写的。

- 特点：

1、实现模板复用。

2、允许嵌套数组、对象进行渲染。

3、保留this关键字（不通过with来访问作用域链）

4、做了部分容错处理，防止未知错误发生

5、预处理data数据，防止内含html标签，预防xss攻击

6、支持AMC、CMD、COMMONJS模块使用。
